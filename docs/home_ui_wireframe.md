# 首页 (Home) UI/UX 交互线稿与视觉规划 (Bento Box 风格)

## 1. 整体视觉风格 (Visual Tone)
*   **风格基调**：**Bento Box (便当盒) 风格**。把各种不同属性的信息块（数据、图文、代码片段）装进圆角的卡片盒子里，形成网格化的大胆排版。
*   **空间感**：卡片之间留有明显的间隙，页面不再是自上而下一大坨字，而是高度模块化的“控制台”。

---

## 2. 页面结构线稿 (Wireframe)

### 2.1 巨型数字名片区与核心价值观 (Hero Dashboard)
将传统的 Hero 区设计为一份“结构化人生宣言”，把“追求”与“方法论”像脑图一样解构在首页最显眼的位置。

```text
  JESS.PU
  
  “通过创造实现自由，通过分享放大价值”
  (超大字号 Slogan，作为视觉核心)

  --------------------------------------------------
  [ 核心价值观与方法论 (树状或网格解构) ]
  
  🎯 追求：
  ├─ 创造：做喜欢的事情
  ├─ 自由：追求财富自由与精神自由
  └─ 分享：秉承公开与互助的原则

  ⚙️ 方法论：
  ├─ 行动至上：Action speaks louder.

  └─ 长期主义：做时间的朋友。
  
  ⚔️ 竞争模式 (小米模式)：
  └─ 专注、极致、口碑、快。

```
