# 我的故事专题 (My Story) UI/UX 交互线稿与视觉规划

## 1. 整体视觉风格 (Visual Tone)
*   **风格基调**：**数字花园/生活志 (Digital Garden / Lifestyle)**。不同于 AI 的“极客风”和商业的“研报风”，这里需要**人味儿**和**松弛感**。可以增加更多的留白，照片可以加上轻微的拍立得/胶片边框效果。文字排版可以稍微随性一些，比如引言可以用带有点手写感的斜体字。
*   **排版逻辑**：沿用全站的左导航右内容双栏布局，但右侧的卡片变成了“大图+碎碎念”模式。

---

## 2. 页面结构线稿 (Wireframe)

### 2.1 顶部横幅 (Hero Section)

```text
[ 巨型粗体标题 ] 我的故事与生活
[ 副标题 ] 代码和商业之外的真实生活。记录成长、学习方法、家庭日常，以及一些毫无用处的发呆时刻。
```

### 2.2 核心双栏结构 (Two-Column Layout)

```text
[ 左侧栏 (Dynamic Nav) - 吸顶 ]     [ 右侧主内容区 (Lifestyle Card Stream) ]
                                |
📍 生活侧影 (后台动态配置)         |   ==== 👨‍👩‍👧 家庭与生活 (Family & Life) ====
                                |
- 📖 个人随笔                     |   [ 大画幅封面图：一张阳光下的小孩或家庭合照 ]
- 👨‍👩‍👧 家庭与生活                 |   [ 📅 2026-02-28 | 📹 Vlog ]
- 📹 日常 Vlog                    |   主标题：周末去山里放空的两天
- 🧠 学习与成长                   |   [✍️ 碎碎念]: "平时盯着屏幕太久了，周末决定断网48小时。大自然的白噪音比任何 App 都管用..."
                                |   [ 观看 Vlog ↗ ]
                                |   
                                |   ==== 🧠 学习与成长 (Learning & Growth) ====
                                |
                                |   [ 无图/或小图：展示书桌一角的照片 ]
                                |   [ 📅 2026-02-15 | 📖 Article ]
                                |   主标题：我是怎么把看美剧变成“真·学英语”的？
                                |   [✍️ 核心方法]: 结合了 Anki 和 YouTube 字幕提取插件，一套极其硬核但好用的工作流分享。
                                |   [ 阅读全文 ↗ ]   |   [ ☕️ 赞赏此文 ]
```

### 2.3 卡片细节差异 (Card Differentiation)
*   **与商业分析的区别**：商业分析卡片没有图，全是干瘪的论点；这里的卡片如果有图（比如 Vlog），图要占主要视觉面积；如果有文字，文字要是感性的“碎碎念 (Snippet)”，而不是冰冷的“Takeaways”。
